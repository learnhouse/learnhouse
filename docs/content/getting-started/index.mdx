# Getting Started

Welcome to LearnHouse — an open-source learning management system for creating and delivering modern educational content.

This section will help you go from zero to a working LearnHouse setup, whether you choose our managed cloud platform or a self-hosted deployment.

## What you will learn

- **[Quickstart](/getting-started/quickstart)** — Get up and running in minutes with LearnHouse Cloud or the self-hosted CLI.
- **[Cloud vs Self Hosting](/getting-started/cloud-vs-self-hosting)** — Understand the trade-offs so you can pick the right deployment model.
- **[Key Concepts](/getting-started/key-concepts)** — Learn the core building blocks: organizations, courses, activities, collections, trails, and the editor.

<Callout>
If you already know how you want to deploy, jump straight to the [Quickstart](/getting-started/quickstart).
</Callout>
