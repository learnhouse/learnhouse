# Maintenance Overview

Keeping your self-hosted LearnHouse instance healthy requires periodic maintenance. The LearnHouse CLI provides built-in tools for the most common maintenance tasks.

## Maintenance Tasks

### [Updates](/self-hosting/maintenance/updates)

Keep LearnHouse up to date with the latest features, improvements, and security patches using `learnhouse update`.

### [Backups](/self-hosting/maintenance/backups)

Protect your data with regular backups of the database and uploaded content using `learnhouse backup`.

### [Monitoring](/self-hosting/maintenance/monitoring)

Monitor the health and performance of your instance using `learnhouse status` and `learnhouse logs`.

### [Troubleshooting](/self-hosting/maintenance/troubleshooting)

Diagnose and resolve common issues with your self-hosted instance.
