# SCORM

<Callout>
SCORM support requires an Enterprise plan.
</Callout>

LearnHouse supports importing SCORM (Sharable Content Object Reference Model) packages — the industry standard for e-learning content. Import content from authoring tools like Articulate Storyline, Adobe Captivate, or iSpring and run it directly within your courses.

## Supported Versions

- **SCORM 1.2** — The most widely supported version.
- **SCORM 2004** — Newer version with enhanced sequencing.

## Importing Packages

1. Upload a SCORM `.zip` package to a course.
2. LearnHouse analyzes the package and detects the version, title, and SCOs.
3. Assign SCOs to chapters and import — each SCO becomes a course activity.

You can also import a package as a new course, automatically creating the course structure from the package contents.

## Tracking

Imported SCORM content reports progress back to LearnHouse, including completion status, scores, time spent, and bookmarking for resume support.
