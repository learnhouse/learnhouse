# Enterprise Edition

The LearnHouse Enterprise Edition (EE) is a commercially licensed extension of the open-source platform. It includes additional features designed for organizations that need advanced identity management, compliance tooling, and support for industry-standard learning content formats.

## Licensing

The Enterprise Edition is available under a separate **EE license**. It is not covered by the open-source license that governs the core platform.

- **Self-hosted** — The EE features are included in the codebase under `apps/api/ee/` but are locked behind an Enterprise license. You need a valid license key to activate them.
- **LearnHouse Cloud** — Some EE features (like Payments) are available on lower-tier plans. Others (SSO, SCORM, Audit Logs) require an Enterprise plan.

Interested in the Enterprise Edition? Visit [learnhouse.app](https://www.learnhouse.app) to learn more about pricing and licensing.

## Enterprise Features

- **[Single Sign-On (SSO)](/enterprise/sso)** — Authenticate users through WorkOS (which brokers SAML, Okta, Azure AD, and more) or any OIDC-compatible provider via the Custom OIDC integration.
- **[Audit Logs](/enterprise/audit-logs)** — A full audit trail of all platform activity for compliance and security monitoring, with filtering and CSV export.
- **[SCORM](/enterprise/scorm)** — Import and run SCORM 1.2 and 2004 learning packages from authoring tools like Articulate Storyline and Adobe Captivate.
