# AI for Teachers

The LearnHouse AI assistant helps teachers create and manage course content more efficiently.

## Content Creation Assistance

Teachers can use the AI to accelerate the course-building process:

- **Draft generation** — Generate initial drafts for activity content based on a topic or outline.
- **Content improvement** — Refine existing content for clarity, tone, or completeness.
- **Quiz creation** — Generate quiz questions based on the course material.

## Course Management

Beyond content creation, the AI can assist with organizing and structuring courses:

- Suggest how to break a topic into chapters and activities.
- Identify gaps in course coverage.
- Recommend improvements based on the existing content structure.

## Getting Started

The AI assistant is available from the course editor interface. Make sure your instance has a Google Gemini API key configured via `LEARNHOUSE_GEMINI_API_KEY` (see [AI Setup](/self-hosting/configuration/ai-setup)) and that AI is enabled for your organization.
