# Courses

Courses are the core content unit in LearnHouse. Each course is made up of chapters, and each chapter contains activities that students work through.

<br/>
<img style={{borderRadius:10}} quality={100} src="/img/pages/courses/courses.png" alt="Courses overview" width={800} height={1300} />

## Activity Types

LearnHouse supports several activity types to accommodate different teaching styles:

- **Dynamic Pages** — Rich WYSIWYG content built with the block-based editor (quizzes and code blocks can be embedded inline).
- **Videos** — Upload video files directly or embed from YouTube and other external sources.
- **Documents** — PDF documents displayed with a built-in viewer.
- **Assignments** — Gradable tasks such as file uploads and quizzes. See [Assignments](/platform/assignments).
- **SCORM** — SCORM 1.2 and SCORM 2004 learning packages (Enterprise Edition).
- **Custom** — Escape hatch for custom activity implementations.

## Getting Started

To create a course, navigate to your organization dashboard and select **Create Course**. From there you can add chapters, populate them with activities, and publish when ready.

<Callout>
Courses can be grouped into [Collections](/platform/courses/collections) for better organization, and student progress is tracked via [Trails](/platform/courses/trails).
</Callout>
