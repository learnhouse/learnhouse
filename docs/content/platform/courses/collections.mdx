# Collections

Collections let you group related courses together, making it easier for students to discover and navigate structured learning paths.

## Use Cases

- **Learning Paths** — Arrange courses in a recommended sequence so students know what to take and in what order.
- **Curricula** — Bundle all courses for a program, department, or certification into a single collection.
- **Categorization** — Organize courses by topic, difficulty level, or audience.

## How Collections Work

A collection is simply a named group of courses. Each course can belong to one or more collections. When students browse a collection, they see all the included courses in the order you define.

To create a collection, go to your organization dashboard and select **Collections**. From there you can name the collection, add courses, and arrange their order.
