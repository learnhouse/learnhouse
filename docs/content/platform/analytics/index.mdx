# Analytics

LearnHouse provides analytics to help administrators and teachers understand how learners engage with content. Analytics track user activity, course engagement, and platform usage.

<Callout>
Analytics must be enabled in your [Organization Settings](/platform/organizations/settings) to be available.
</Callout>

## What's Tracked

- **User engagement** — Activity views, time spent, and interaction patterns.
- **Course analytics** — Which courses and activities are most popular and how students progress through them.
- **Organization-level metrics** — Overall platform usage and growth trends.

## How Analytics Work

Events are tracked as users interact with the platform — viewing activities, completing courses, submitting assignments, and more. These events are aggregated into dashboards accessible from the admin panel.

## Accessing Analytics

Administrators can access the global analytics dashboard from the organization admin panel. Analytics data is scoped to the current organization, so each organization sees only its own data.

<Callout>
Analytics uses event caching for performance. Data may take a short time to reflect the most recent activity.
</Callout>
