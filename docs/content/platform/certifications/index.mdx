# Certifications

LearnHouse can generate certificates when students complete a course, providing formal recognition of their learning achievements.

## How Certifications Work

When a student completes all activities in a course, a certification record is created and linked to their profile. Teachers and administrators can manage certifications at the course level.

## Key Features

- **Automatic generation** — Certificates are generated when a student's Trail shows full course completion.
- **Course-level configuration** — Each course can have certifications enabled or disabled.
- **Certificate retrieval** — Students can view and access their earned certificates from their profile.
- **Verification** — Each certificate has a unique identifier that can be used for verification.

## Use Cases

- **Professional development** — Issue certificates for compliance training or continuing education.
- **Academic programs** — Provide completion certificates for course series or learning paths.
- **Skill validation** — Give students a shareable credential they can reference externally.
