# Groups & Bundles

Groups let you bundle multiple courses together and sell them as a single package. When a student purchases an offer linked to a group, they get access to all courses in that group.

## How Groups Work

1. Create a group and give it a name.
2. Add courses to the group.
3. Link the group to one or more offers.
4. When a student purchases the offer, they are enrolled in all courses within the group.

## User Group Sync

Groups can be synced with user groups for automatic access provisioning. When a payment group is synced to a user group, students who purchase an offer linked to that group are automatically added to the corresponding user group, granting them access to all associated resources.

## Direct Resource Linking

If you don't need a group, you can also link individual courses directly to an offer. This is useful for selling single courses without creating a bundle.
