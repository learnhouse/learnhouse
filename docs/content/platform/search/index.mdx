# Search

LearnHouse includes a built-in search system that lets users find courses, collections, and other users across the organization.

## How Search Works

Search is scoped to the current organization. When a user enters a query, the system searches across:

- **Courses** — Match by course title, description, and content.
- **Collections** — Find collections by name or description.
- **Users** — Look up organization members by name or username.

Results are returned with pagination, displaying up to 50 results per page.

## Features

- **Organization-scoped** — Each organization's search results are isolated, ensuring users only see content from their own organization.
- **Cross-content search** — A single search query returns results across multiple content types.
- **Pagination** — Results are paginated for efficient browsing of large result sets.
