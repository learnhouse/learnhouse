# Editor

LearnHouse includes a block-based WYSIWYG editor for creating course content. The editor provides a Notion-like experience where you build pages by stacking and arranging content blocks.

## Key Features

- **Block-based editing** — Each piece of content (paragraph, image, code snippet) is an independent block that you can move, duplicate, or delete.
- **Rich formatting** — Full text formatting including bold, italic, links, and inline code.
- **Drag and drop** — Rearrange blocks by dragging them into the desired position.
- **Slash commands** — Type `/` to quickly insert any block type.
